## Step 4 — Content Production Rules

Every piece of content must carry three tags:
```
[Pillar]     P1 / P2 / P3 / P4 / P5
[Sector]     Health / Green / General
[Direction]  N→T / T→N / Both
```

Before drafting any content ask: "Which named profile would stop scrolling for this — A or B?"
If the answer is "neither" or "everyone" — rewrite the opener.
