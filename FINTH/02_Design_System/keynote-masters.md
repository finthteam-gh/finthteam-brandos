## CoWork AppleScript Instructions
When generating a Keynote presentation via CoWork:
1. Always open `02_Design_System/brand-template.key` as the base file
2. Apply master layouts by their exact names listed above
3. Populate text placeholders with generated content
4. Save output to `05_Output_Archive/YYYY-MM-DD_presentation_[description].key`

## Presentation Colour Rules
- Primary background: warm off-white `#F5F3EF`
- Dark background slides: deep forest `#2E4A3E`
- Accent: warm sand `#9C7A52`
- Never use pure white or pure black

## Font on Slides
- All text: brand font (see typography.md)
- Never override with system fonts
