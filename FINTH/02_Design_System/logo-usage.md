# Logo Usage — FINTH

## Name Format
Always written: **FINTH** — all capitals, always.
Never: Finth, finth, F.I.N.T.H.

## Logo Placement
- Default position: bottom-right corner on all image assets
- Size: small, understated — never dominant
- Clear space: maintain breathing room around the wordmark

## Logo on Backgrounds
- On warm off-white (`#F5F3EF`): use deep forest (`#2E4A3E`) version
- On deep forest (`#2E4A3E`): use warm off-white (`#F5F3EF`) version
- On warm sand (`#9C7A52`): use deep forest version

## What Not to Do
- Never stretch or distort the logo
- Never add drop shadows or effects
- Never place on a busy photographic background without an overlay
- Never use a colour version not specified above
- Never make it the hero element of a design — FINTH leads with ideas, not branding

## File Locations
Logo files should be stored in: `_SHARED/Logos/FINTH/`
Formats needed: .svg (master), .png (transparent background), .pdf (print)
