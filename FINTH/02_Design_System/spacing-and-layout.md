# Spacing and Layout — FINTH Design System

## Core Principle
Minimal. Quiet. Let the content breathe. FINTH's aesthetic is editorial — the kind of restraint that signals confidence, not lack of ideas.

## Layout Rules

**Whitespace:** Use generously. Overcrowded layouts are anti-FINTH.
**Alignment:** Left-aligned text as default. Centre only for cover slides or hero moments.
**Grid:** Simple. No complex multi-column layouts unless content genuinely requires it.
**Hierarchy:** Clear visual weight difference between headings and body. Never ambiguous.

## Document Layout (Word/Pages)

- Margins: generous (min 2.5cm all sides)
- Line spacing: 1.4–1.6 for body text
- Paragraph spacing: add space after paragraphs, not between lines
- Headers: clearly distinct from body — size and weight difference

## Slide Layout (Keynote)

- One key idea per slide — never crowded
- Text: never fill the slide. Use 60% of available space maximum.
- Images: full-bleed or contained — never awkward in-between sizes
- Avoid: clip art, decorative borders, drop shadows, gradients

## Social Media Layout

- LinkedIn square (1080×1080): text occupies bottom third, image or background fills top
- LinkedIn landscape (1200×627): text left-aligned, image right or full bleed with overlay
- Text on image: maximum 8 words, clean bold sans-serif, high contrast

## Print / PDF

- A4 format standard
- Bleed: 3mm if going to print
- Font embedding: always embed — never rely on system substitution
