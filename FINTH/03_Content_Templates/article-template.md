# Article / Blog Post Template — FINTH

## Format Standards

- Length: 800-1,500 words for articles / 500-900 words for blog posts
- Structure: prose paragraphs — no bullet lists unless genuinely list-like
- Sentence case throughout
- No em dashes

## Required Tags

```
[Pillar]     P1 / P2 / P3 / P4 / P5
[Sector]     Health / Green / General
[Direction]  N→T / T→N / Both
[Profile]    A / B
[Type]       Article / Blog
[Word count] [actual count]
```

## Structure

```
# [Title — sentence case, contains the central argument]

[Opening paragraph — 2-3 sentences. State the conclusion first. No preamble.]

## [Section heading — sentence case]

[Body paragraphs — each paragraph makes one point. Lead with the point, support with evidence.]

## [Section heading]

[Continue...]

## [Final section — what this means for the reader]

[Closing paragraph — 1-2 sentences. A concrete implication or call to think differently. No summary, no "in conclusion".]

---
[Pillar] / [Sector] / [Direction] / [Profile]
Generated under: FINTH Brand v[X.X] — [Date]
```

## Pre-Write Checklist

- [ ] Which named profile is this for — A or B?
- [ ] Central argument stated in opening paragraph
- [ ] Every claim sourced or flagged for verification
- [ ] No motivational language disconnected from the corridor
- [ ] Tone: clear, thoughtful, realistic
- [ ] No em dashes, no banned words
