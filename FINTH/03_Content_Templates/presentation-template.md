# Presentation Template — FINTH

## Base File
Always start from: `02_Design_System/brand-template.key`
Never start from a blank file.

## Standard Deck Structure

1. Cover slide — FINTH name, presentation title, date
2. Context / situation — what we are responding to
3. Recommendation / approach — the main point, stated clearly
4. Evidence — data, corridor research, case examples
5. Operational detail — how it works, timeline, who does what
6. Investment — fees, retainer, scope
7. Next steps — clear, dated, owned

## Slide Rules
- One key idea per slide
- Text: 60% of slide maximum — let it breathe
- Data visualised where possible, not listed
- All fonts: brand font only
- All colours: from FINTH colour palette only

## Client Presentation Naming
Save to: `CLIENTS/[CLIENT NAME]/04_Deliverables/`
File name: `YYYY-MM-DD_[client]_[deck-type].key`
