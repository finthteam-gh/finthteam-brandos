# Blog Post Template — FINTH

## Metadata Block (Top of Every Post)
```
Title: [Sentence case, contains focus keyphrase, factual]
Focus keyphrase: [primary keyword]
Meta description: [150–160 chars, factual, no hype]
Pillar: [P1/P2/P3/P4/P5]
Sector: [Health/Green/General]
Direction: [N→T / T→N / Both]
Profile target: [A / B]
```

## Structure
- H1: Article title
- Opening paragraph: state the main argument immediately
- H2 sections: each covers one distinct point
- Body: prose paragraphs — not bullet lists unless genuinely list-like
- Data: always attributed, never invented
- Close: one clear recommendation or forward-looking point

## Word Count
Standard article: 800–1,200 words
Deep feature: 1,200–2,000 words

## Pre-Flight Check
- [ ] Profile target named and content genuinely serves them
- [ ] All data attributed or flagged for verification
- [ ] No em dashes, sentence case throughout
- [ ] Tone: clear, thoughtful, realistic
- [ ] No banned words or vague Asia statements
