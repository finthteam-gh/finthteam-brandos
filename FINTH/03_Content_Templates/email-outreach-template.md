# Email / DM Outreach Template — FINTH

## Outreach Principles

- Never pitch in the first message
- 3-5 sentences maximum for first contact
- Always reference something specific to the person
- Always write as the CEO — personal, direct, warm but not familiar
- Thai contacts: warmth before business, always

## Message Types

1. LinkedIn connection request note (300 characters max)
2. First DM after connection (no pitch, genuine observation)
3. Follow-up DM after they engage with content (reference the specific post)
4. Re-engagement message (connection gone quiet — 90 days no interaction)
5. Warm brief send note (sharing the quarterly corridor brief)
6. Partner introduction request (asking a mutual contact for introduction)

## Draft Format

```
Contact type: [Nordic executive / Thai business leader / potential partner / journalist]
Message type: [Type 1-6]
Context: [1 line on what you know about this person]
Draft: [the message]
Tone check: [confirm it passes: clear, not salesy, specific]
```

## Example — Type 2 First DM (Profile A)

```
Contact type: Nordic executive — Finnish healthtech commercial director
Message type: Type 2 — first DM after connection
Context: Just accepted connection, has been posting about SEA distribution challenges

Draft:
"Saw your post about the distribution challenges in SEA last week — the point about finding partners who actually understand cold chain in Thailand resonated. That gap between what's promised in the pitch and what happens at the warehouse is something I work on specifically with Finnish brands entering the market. Happy to share what I've learned if it's useful."

Tone check: Personal reference to specific post ✓ / No pitch ✓ / Specific not generic ✓ / Under 5 sentences ✓
```
